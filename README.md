# Groep1ProjectD

### Docker command for running the containers

```
docker compose up -d
```

### Docker command for stopping the containers

```
docker compose down
```

### installation

```
pip install -r requirements.txt
conda install -c conda-forge geckodriver
```

### Running Frontend

```
streamlit run main.py
```

### Running Backend

```
python backend.py
```

### Running the tests
```
python -m unittest
```
